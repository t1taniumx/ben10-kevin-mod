Ben 10: Power Trip — Kevin Mod
Replaces Ben with Kevin Levin as the playable character. Co-op works with this version.
Note: UI icons are incorrect. This is a known limitation and does not affect gameplay.

Installation

1. Download BepInEx and extract it into your game folder (where the .exe is) 
   link: https://github.com/BepInEx/BepInEx/releases

2. Launch the game once and close it

3. Download the mod and place the .dll file into BepInEx/plugins

4. Launch the game and you are Kevin
